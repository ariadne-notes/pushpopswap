# ISE and PxGrid

Version 1 used XMPP.

Version 2 uses HTTPS-only REST.

PxGrid is used with Catalyst Center.

## Terms

**pxGrid** --- Platform Exchange Grid

**Controller**

- Discovery
- Authentication
- Authorization

**Provider**

- Returns query results
- Publishes

**Pubsub**

- Provides pxGrid Services

**Subscriber**

- Can subscribe to topics
  - Alerts
  - Context Information
