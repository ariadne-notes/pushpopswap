# Secure VPNS

**IPSec**

IPSec with IKEv2 in Tunnel mode is an Industry Standard. 

- OK for point-to-point
- No Dynamic Routing
- No Multicast

**GRE IPSec**

- Tunnels Multicast

**GETVPN**

- Cisco Proprietary
- One SA for the entire group
- Native Multicast

**DMVPN**

- Automates the setup of point-to-point SAs.
- Works over the Internet
- Tunnels Multicast

**SD-WAN**

- Controller based
- Supports ZTP
- Managed via webpage
- Tunnels Multicast
