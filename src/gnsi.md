# gNSI

**gNSI** -- gRPC Network Security Interface

## Functions

**Certz**

- Certificate Management
- CRL management

**Acctz**

- Accounting records

**Pathz**

- Access control for gNMI paths

**Authz**

- Authorization for users to access gRPC services
