# Subnetting with the Box Method

For visual learners.

For a `/24` network, thats 256 addresses, or ... this box.

## Start with A /24

Write the first IP in the top corner, the last IP in the bottom corner.

```bob
┌─────────────────────────────────┐
│0                                │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                                 │
│                              255│
└─────────────────────────────────┘
```
