# Hextet Boundaries

For any of these, to subnet them, just change the final hex digit.

```text
/64   FFFF:FFFF:FFFF:FFFF::
/128  FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF:FFFF
```
