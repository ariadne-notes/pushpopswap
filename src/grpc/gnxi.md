# gNxI

**gNxI** --- gRPC Network X Interface

Usually combines the gNMI and gNOI tools.

## Unsecure Setup

### Config

```console,editable
gnxi
gnxi server
gnxi port <port number>
```

## Verify

```console,editable
show gnxi state
```
