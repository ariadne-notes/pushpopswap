# TrustSec

Using tags to represent group privilege.

## Phases

1. Classification

A user connects, and a tag is assigned.

2. Propagation

**Inline tagging**

- Traffic itself caries the SGT

**SXP** --- SGT Exchange Protocol

3. Enforcement

- Routers
- Switches
- Firewalls
